flexModal Examples

Simple Examples

  • Close button removed

    data-modal='{"closebutton" : "false"}' Click Me

  • Close button Present

    data-modal='{"closebutton" : "true"}' Click Me

  • Changed the overlay.

    data-modal='{ "overlay": "0.1"}' Click Me

  • Sizing Auto

    data-modal='{"sizing" : "auto"}' Click Me

  • Src is set in data-modal

    data-modal='{"src" : "#content"}' Click Me

  • Src set in data-modal, top also set

    data-modal='{"top": "100px", "src" : "#content"}' Click Me

  • An Href and a seamless iframe

    href="http://nationwide.com" data-modal='{"iframe": {"seamless": "true"} }' Click me

  • Multiple Properties

    { "top": "20px", "overlay": "0.5", "sizing": { "height" : "700", "width" : "500" }, "src" : "#content" } Click Me
